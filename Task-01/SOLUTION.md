**GH COMMANDS USED IN THIS TASK:**
## git add .:
The git add command adds a change in the working directory to the staging area. It tells Git that you want to include updates to a particular file in the next commit.
## gh auth login:
 Authenticate with a GitHub host. The default authentication mode is a web-based browser flow.
## gh commit "commit msg":
Used to commit changes to the repo. when used along with -m, it commits changes to the main branch.
## gh pull origin master :
will pull changes from the origin remote, master branch and merge them to the locally checked-out branch.
